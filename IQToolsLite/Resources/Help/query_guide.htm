IQTools hides the complexity of EMR databases by creating easy to use tables and views that combine multiple database tables to present the data in a user friendly format.

The same structure is applied across multiple EMRs used in different countries and programs therefore providing a standard environment for designing queries and extracting patient data. Since the database structures are different for the different EMRs, this is necessary to lessen the burden of learning database structures on the user.
