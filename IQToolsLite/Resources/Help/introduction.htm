IQTools is a robust data analysis and reporting application developed by Futures Group. Deployed in over 200 sites spanning 8 countries, IQTools provides a user friendly interface for data analysis through creation of SQL (Structured Query Language) queries.

IQTools supports multiple EMRs (Electronic Medical Records Systems). IQCare in Kenya, Uganda and Nigeria. CTC2 in Tanzania. IQChart in Rwanda and Guyana. ISante in Haiti.

IQTools Lite has been developed to provide easy querying of de-identified data from various countries through a secured connection over the internet.
